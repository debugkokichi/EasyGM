# EasyGM
Easy Gamemode (in short "EasyGM") is a gamemode plugin, wich allows you to change your gamemode pretty easy.

## Commands
> Gamemode 0 --> /gm0
> Gamemode 1 --> /gm1
> Gamemode 2 --> /gm2
> Gamemode 3 --> /gm3

## Perms
> Gamemode 0 --> gm0.cmd
> Gamemode 1 --> gm1.cmd
> Gamemode 2 --> gm2.cmd
> Gamemode 3 --> gm3.cmd
