# EasyGM
Easy Gamemode (in short "EasyGM") is a gamemode plugin, wich allows you to change your gamemode pretty easy.

## Commands
1. Gamemode 0 --> /gm0
2. Gamemode 1 --> /gm1
3. Gamemode 2 --> /gm2
4. Gamemode 3 --> /gm3

## Perms
1. Gamemode 0 --> gm0.cmd
2. Gamemode 1 --> gm1.cmd
3. Gamemode 2 --> gm2.cmd
4. Gamemode 3 --> gm3.cmd
