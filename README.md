# EasyGM
Easy Gamemode (in short "EasyGM") is a gamemode plugin, wich allows you to change your gamemode pretty easy.

## Commands
<!-- 0 --> > Gamemode 0 --> /gm0
<!-- 1 --> > Gamemode 1 --> /gm1
<!-- 2 --> > Gamemode 2 --> /gm2
<!-- 3 --> > Gamemode 3 --> /gm3

## Perms
<!-- 0 --> > Gamemode 0 --> gm0.cmd
<!-- 1 --> > Gamemode 1 --> gm1.cmd
<!-- 2 --> > Gamemode 2 --> gm2.cmd
<!-- 3 --> > Gamemode 3 --> gm3.cmd
